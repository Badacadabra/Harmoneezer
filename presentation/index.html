Technique

Environnement de travail

  • Gestionnaires de paquets : npm & Bower

  • Workflow avec task runner : Gulp
  • Modularisation et empaquetage : Browserify
  • Versioning : Git & GitHub

Conception & Développement

  • Introduction à JavaScript
  • Programmation orientée objet (POO)
  • Design patterns utilisés :
    Singleton, Factory, Facade, Adapter, Iterator, Strategy, Module, MVVM
  • Interrogation de deux APIs RESTful :
    Deezer & Echo Nest (Spotify)
  • Création d'interface graphique avec Semantic UI
  • Balisage sémantique avec métadonnées : Open Graph, Twitter Cards, Microdata
  • Documentation des modules avec YUIDoc
  • Tests :
    - unitaires avec QUnit
    - fonctionnels avec CasperJS & PhantomJS
    - de performance avec Benchmark.js
